About ShopMath

ShopMath / MachinCalc is a browser-based machine shop calculator suite for daily shop-floor setup and inspection work. It is built as a single static HTML file with vanilla JavaScript, so it can run locally or from GitHub Pages without a server.

Live app:
https://universepattern.github.io/shopmath/
Repository:
https://github.com/universepattern/shopmath

Calculations are intended as practical shop helpers. Always verify critical dimensions, feeds, speeds, and thread data against your tooling, material, machine condition, and shop standards before cutting metal.
