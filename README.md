# ShopMath / MachinCalc

A single-file Machine Shop Calculator Suite for daily shop-floor calculations.

## Tools

- Degree to DMS converter
- Scientific calculator
- PCD hole position calculator
- Drill point depth calculator
- Taper calculator
- Thread tap drill calculator
- Surface speed / RPM calculator
- Learn how to use section

## Run locally

Open `index.html` in a browser.

## GitHub Pages

This is a static app and can be hosted directly from the repository root using GitHub Pages.

Live app: https://universepattern.github.io/shopmath/
